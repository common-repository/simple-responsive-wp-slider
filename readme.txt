== Description ==

The simplest, most lightweight responsive slider plugin for WordPress. Create and customize unlimited sliders. Place your sliders with Gutenberg blocks or use a shortcode. Place multiple sliders on each page.

Some of the awesome features include:

*   Create unlimited sliders
*   Create multiple sliders on each page
*   Use shortcodes or blocks to add your sliders 
*   Set custom configuration options for each slider, including navigation and title

== Installation ==

1. Unzip the plugin file
2. Upload the folder `simple-wp-responsive-slider` and its contents to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Use the `Slider Builder` plugin menu to create a new  `Slider`
5. Copy the Shortcode (ex. `[sws_slider id="35" title="your new slider"]`) next to your new list and embed it anywhere in your website.
6. Happy slider building!

== Frequently Asked Questions ==

= Can I configure my slider options? =

Yes! Basic configuration includes show / hide arrows and dots from individual sliders.
